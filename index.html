DataMatrix

Este projeto utiliza Terraform, uma ferramenta de código aberto que permite definir, provisionar e gerenciar infraestrutura como código (IaC). Com o Terraform, automatizamos a criação e manutenção de recursos no Google Cloud Platform (GCP), garantindo que a infraestrutura seja replicável e consistente.

  • Push no GitHub: Alterações são feitas no branch principal.
  • Checkout: O código é clonado para a máquina virtual do GitHub Actions.
  • Configuração do Terraform: A versão específica do Terraform é instalada.
  • Credenciais do GCP: As credenciais são configuradas para autenticação com o GCP.
  • Inicialização do Terraform: Prepara o diretório de trabalho para o Terraform.
  • Validação: Verifica se a configuração é válida.
  • Formatação: Ajusta o formato do código para seguir as convenções.
  • Planejamento: Mostra as alterações que serão aplicadas.
  • Aplicação: Aplica as alterações na infraestrutura do GCP.
  • Carregamento de Dados: Executa um script para carregar dados no BigQuery.

Data_Lk

Solução completa e integrada para o gerenciamento, versionamento e sincronização de bancos de dados, ideal para projetos pessoais e portfólios. Essa Solução combina tecnologias modernas – DuckDB, GitHub e Supabase – para oferecer uma experiência robusta, escalável e totalmente gratuita.

  • Solução Completa e Gratuita!
  • Operações CRUD.
  • Criação Dinâmica de Tabelas.
  • Sincronização de Esquemas e Dados.
  • Carregamento Incremental de Dados.
  • Execução Paralela e Eficiência.
  • Robustez e Monitoramento.
  • Integração com PostgreSQL (Supabase).
  • Flexibilidade e Facilidade de Uso.
  • Rastreabilidade e Controle de Alterações.
  • Desempenho e Escalabilidade.
  • DuckDB, GitHub e Supabase.
